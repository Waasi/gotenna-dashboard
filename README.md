# GoTenatious Mesh Dashboard

GoTenatious Mesh emergency feed dashboard

## Requirements

- [Docker](https://www.docker.com/)

## Setup

1. Copy `example.env` as `.env`
2. Add `MONGO_HOSTNAME` value
3. Add `BASE_URL` value
4. Add `DB_NAME` value
5. Add `PORT` value
6. Run `make build`

## Run

To run the app do: `make up`
